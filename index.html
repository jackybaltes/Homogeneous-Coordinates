pt = [5,3]

theta = 30.0/180.0 * math.pi

ax.text( math.cos(alpha/2) * 0.5, math.sin(alpha/2)*0.5, "α")
ax.add_artist( alphaArtist)

ax.plot( [ pt[0], pt[0] ], [0, pt[1]], 'r--', linewidth=2 )
ax.text( pt[0], 0.1, "x" )

ax.text( 0.1, pt[1], "y" )

ax.plot( [0,math.cos(alpha)], [0,math.sin(alpha)], 'b-' )
ax.plot( [0,math.cos(alpha + theta)], [0,math.sin(alpha + theta)], 'b-' )

Rotation

Rotations

Rotation of the shape around the origin (0,0)

Given is a point \( Pt \) with coordinates \((x,y)\)

After rotation by \( \theta \) around the origin, what are the new coordinates of the rotated point \( Pt' \)?

Find a function \( f_\theta(x,y) => x',y' \)

Derivation of the Rotation Formula

Convert the Cartesian coordinates \( (x,y) \) into polar coordinates \( ( \alpha, r ) \)

\[ \begin{eqnarray} r = \sqrt{ x^2 + y^2 } \\ \alpha = arctan(y,x) \end{eqnarray} \]

Note that to calculate the arctan, you want to use atan2(y,x) (Python, C, C++, ... ) instead of the atan( y/x )

After Rotation by θ

The distance to the origin is unchanged. The angle increased by \( \theta \)

\[ \begin{eqnarray} r' = r\\ \alpha' = \alpha + \theta \end{eqnarray} \]

Convert from polar to cartesion coordinates

\[ \begin{eqnarray} x' = r' * \cos(\alpha') = r * \cos(\theta+\alpha)\\ y' = r' * \sin(\alpha') = r * \sin(\theta+\alpha)\\ \end{eqnarray} \]
